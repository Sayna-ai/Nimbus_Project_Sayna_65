# Hospital Prescription and Billing System  
### Project Number: 65  
### Repository: Nimbus_Project_Sayna_65

---

## 1. Project Objective

The objective of this project is to develop a Hospital Prescription and Billing System using the C programming language.  
The system helps clinics and hospitals record patient details, store prescribed medicines, compute billing amounts, apply tax, and generate an invoice automatically.  
This project demonstrates real-world use of:

- Structures  
- Arrays  
- Functions  
- Pointers  
- Input validation  
- File handling  
- Modular programming  

---

## 2. Development Steps

### **Step 1: Create Structures**
- `Medicine` structure to store name, dosage, price, quantity, amount  
- `Patient` structure to store ID, name, age, gender, phone  
- `Prescription` structure to store patient, medicines, subtotal, tax, total

### **Step 2: Implement Input Functions**
- Safe input functions using `read_line_stdin`, `read_int_strict`, `read_float_strict`  
- Strict input validation to prevent invalid entries  
- Patient details handled via `inputPatientDetails()`  
- Medicines added using `addMedicines()`  

### **Step 3: Billing Logic**
- Subtotal computed by summing medicine amounts  
- Tax applied using constant `TAX_RATE (18%)`  
- Total calculated in `calculateBill()`  

### **Step 4: Generate Output**
- `printInvoice()` displays a formatted bill  
- `saveInvoiceToFile()` stores invoice in `invoice.txt`  

### **Step 5: Compile and Test in Nimbus**
- Compile using:
bash
gcc main.c -o hospital
