# Hospital Prescription and Billing System  
### Project Number: 65

## 1. Project Objective
The aim of this project is to develop a simple Hospital Prescription and Billing System using the C programming language.  
The system allows hospitals and clinics to enter patient details, add prescribed medicines, calculate billing amounts, apply tax, and generate a complete invoice.  
The project demonstrates key C programming concepts such as structures, arrays, functions, pointers, decision statements, loops, and file handling.

---

## 2. Features
- Store patient information (ID, Name, Age, Gender, Phone)
- Add multiple medicines
- Store medicine details (Name, Dosage, Price, Quantity)
- Automatically calculate:
  - Medicine-wise amount  
  - Subtotal  
  - Tax  
  - Final total
- Generate a formatted invoice
- Save invoice to a text file
- Modular and structured code using functions

---

## 3. C Concepts Used
- **Structures**
- **Arrays**
- **Functions**
- **Pointers**
- **Loops and Conditional Statements**
